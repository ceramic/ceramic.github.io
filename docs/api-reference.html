Browser Windows

The core API is available on the ceramic package.

window
A browser window.
  • id
    A unique string ID for the window.
  • title
    The window title.
  • url
    The window URL.
  • x
    Window's offset from the left.
  • y
    Window's offset from the top.
  • width
    The window's width, in pixels.
  • height
    The window's height, in pixels.
  • resizablep
    Whether or not the window is resizable.
make-window(&key title url x y width height resizablep)
Create a window.
show-window((window window))
Show the window.
hide-window((window window))
Hide the window.
close-window((window window))
Close the window.
destroy-window((window window))
Forcefully close the window.
maximize-window((window window))
Maximize the window.
unmaximize-window((window window))
Un-maximize the window.
minimize-window((window window))
Minimize the window.
unminimize-window((window window))
Un-minimize the window.
center-window((window window))
Center the window.

Resources

Resource management tools are defined in the ceramic.resource package.

define-resources(system-name nil &rest pairs)
Define resource tags for a system.
resource-directory(resource-tag)
Return the directory associated to a resource tag.
resource(resource-tag pathname)
Return the pathname of a resource relative to the directory of a resource tag.

Releasing

The bundler is defined in the ceramic.bundler package.

bundle(system-name &key bundle-pathname)
Compile the application to an executable, and ship it with its resources.
